🇸🇪 SvenskaCoach

Learn Swedish smarter – one word, one story, one lesson at a time.

SvenskaCoach is an Android application designed to help learners master the Swedish language in a structured, enjoyable, and practical way. Instead of focusing only on vocabulary lists, SvenskaCoach combines words, grammar, quizzes, stories, and progress tracking into one learning experience.

The goal is to make Swedish accessible for beginners while providing enough depth for more advanced learners.

⸻

✨ Features

📚 Vocabulary Trainer

* Learn Swedish words grouped by topic
* Example sentences
* Pronunciation support (planned)
* Personal progress tracking
* Favorite difficult words

⸻

🇸🇪 En / Ett Trainer

One of the most difficult parts of Swedish is knowing whether a noun uses en or ett.

SvenskaCoach includes a dedicated trainer that helps users:

* Practice noun genders
* Learn patterns
* Repeat difficult words
* Build long-term memory through repetition

⸻

📖 Story Library

Read Swedish stories adapted to your language level.

Levels include:

* A1
* A2
* B1
* B2

Each story contains:

* Vocabulary list
* Grammar explanations
* Reading questions
* Translation support (optional)
